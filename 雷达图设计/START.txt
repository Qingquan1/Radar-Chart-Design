========================================
    🚀 雷达图快速启动指南
========================================

📋 第一次使用？跟着这个步骤走！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤 1：查看示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

双击打开：index.html

你会看到：
✓ 7款手机的综合对比雷达图
✓ 每款手机的单独详情图
✓ 可以点击图例显示/隐藏手机
✓ 悬停在图表上可以导出图片

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤 2：修改数据（推荐方式）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方式 A：使用可视化编辑器 ⭐⭐⭐⭐⭐

1. 双击打开：editor.html
2. 在表格中修改数据：
   - 修改标题
   - 添加/删除维度
   - 添加/删除产品
   - 输入评分（0-5分）
3. 右侧实时预览效果
4. 点击"导出配置"按钮
5. 将下载的 config.js 替换到项目文件夹
6. 刷新 index.html

⏱️ 耗时：5分钟

方式 B：直接修改配置文件 ⭐⭐⭐⭐

1. 用记事本打开：config.js
2. 修改 CONFIG 对象中的数据
3. 保存文件
4. 刷新 index.html（按 Ctrl+F5）

⏱️ 耗时：3分钟

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤 3：导出图片
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开 index.html
2. 滚动到"单机详情"区域
3. 鼠标悬停在任意图表上
4. 点击右上角的"导出"按钮
5. 图片自动下载

导出的图片：
✓ 纯雷达图（不含标题）
✓ 620×620 像素
✓ PNG 格式
✓ 深色背景

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤 4：更换产品类型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

从"手机评测"改成"笔记本评测"：

1. 打开 editor.html
2. 点击"加载示例"按钮
3. 选择"2 - 笔记本电脑"
4. 根据需要修改数据
5. 导出配置文件
6. 替换 config.js
7. 刷新 index.html

✅ 完成！代码无需任何修改！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 重要提示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 所有产品必须包含相同的维度
2. 评分范围：0-5 分
3. 建议维度数量：6-10 个
4. 建议产品数量：3-7 个
5. 修改前记得备份数据

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ 常见问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 修改后没有变化？
A: 强制刷新浏览器（Ctrl+F5）

Q: 编辑器在哪？
A: 双击 editor.html 文件

Q: 如何更换产品类型？
A: 只需修改 config.js 中的数据

Q: 可以评测其他产品吗？
A: 可以！手机、电脑、汽车等都可以

Q: 导出的图片在哪？
A: 浏览器的默认下载文件夹
